Using the Access Token

Source: https://flask-jwt-extended.readthedocs.io/en/latest/basic_usage.html
$ export ACCESS="j.w.t"
$ curl -H "Authorization: Bearer $ACCESS" "http:/localhost:5000/api/v1/notifications/"
